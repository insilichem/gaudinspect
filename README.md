# GAUDInspect #

GAUDInspect helps you discover good starting points for a GAUDI essay by providing real-time visual aid. Just load your system and move the poses to locate your *pole position*.

## Usage

1. Open up Chimera and load your system.
2. Launch `Tools> GAUDI> GAUDInspect`.
3. Select the scores you want to inspect and start creating poses. 
4. You can save good poses as checkpoints and generate the corresponding mol2 files.
